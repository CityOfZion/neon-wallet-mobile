# Contributing to Neon Wallet Mobile

We welcome contributions to the Neon Wallet Mobile project! Please follow the guidelines below to get started.

## Getting Started

### Setting Up the Development Environment
1. **Clone the Repository**
    ```sh
    git clone https://github.com/CityOfZion/neon-wallet-mobile
    ```
2. **Install Dependencies**
    ```sh
    cd neon-wallet-mobile
    npm install
    ```
3. **Run Expo**
    ```sh
    npm run dev
    ```

## Requirements for Running on Android

1. **Android Studio**: Make sure you have [Android Studio](https://developer.android.com/studio) installed.
2. **Android SDK**: Install the latest Android SDK via Android Studio.
3. **Emulator or Device**: Set up an Android emulator or connect an Android device via USB.

> Ensure your PATH environment variable includes `adb` (Android Debug Bridge).

4. **Run on Android**:
    ```sh
    npm run android
    ```


Thank you for your contributions!
